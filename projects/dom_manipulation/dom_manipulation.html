Creating HTML Elements in Java Script

The Power of Java Script

Since Websites became more dynamic, simple HTML does not cover all the demands that developers have. HTML is the skeleton, CSS the Make-UP and JavaScript the muscles. Since it allows us to manipulate the DOM Elements in a dynamical way, it is possible to extend any HTML code by adding some Hocus Pocus in Java. Classical programming elements like Arrays and loops are available in Java Script which enables the developer to add different dynamict to their websites. The power ov Java Script is not only in dynamics, since there are many Data-Management tools like JSON. This makes Java Script the choice number one scripting language for borwser sided programming.
